# 状态码编码规范与迁移对照

## 1. 背景

为统一后端接口返回码结构，所有接口响应继续采用 `HTTP 200 + 业务 code` 形式。  
`code=0` 表示成功，其他值表示失败或特殊状态。

## 2. 新编码结构（6 位）

- 第 1 位：定义位
  - `1`：系统内部定义状态码
  - `2`：符合 HTTP 标准状态码
- 第 2-4 位：状态码大类（3 位）
- 第 5-6 位：子分类（2 位，当前默认 `00`）

## 3. 统一转换规则

### 3.1 成功码

- 原始值 `0` 保持不变：`0`

### 3.2 符合 HTTP 标准状态码

- 规则：`2 + 原始三位状态码 + 00`
- 示例：
  - `400 -> 240000`
  - `401 -> 240100`
  - `403 -> 240300`
  - `404 -> 240400`
  - `422 -> 242200`
  - `500 -> 250000`
  - `502 -> 250200`
  - `503 -> 250300`

### 3.3 系统内部定义正数状态码（不符合 HTTP 标准）

- 规则：`1 + 三位补齐 + 00`
- 示例：
  - `1 -> 100100`
  - `22 -> 102200`
  - `999 -> 199900`

### 3.4 系统内部定义负数状态码

- 规则：
  1. 对绝对值取后三位并补齐三位；
  2. 将该三位分类值最高位强制改为 `1`；
  3. 最终编码为 `1 + 分类值 + 00`。
- 示例：
  - `-2 -> 110200`
  - `-22 -> 112200`

## 4. 兼容性与实现约束

- 所有接口错误码统一通过 `normalize_status_code(raw_code)` 进行转换。
- `ApiResponseSchema.error(code=...)` 自动执行转换，避免手工重复改写。
- `HTTPException` 统一由全局异常处理器转换后再输出，禁止直接返回裸 HTTP 状态码。
- 已是 6 位规范码（`100000~299999`）时不重复转换。

## 5. 常见旧码到新码对照

- `400 -> 240000`
- `401 -> 240100`
- `403 -> 240300`
- `404 -> 240400`
- `409 -> 240900`
- `422 -> 242200`
- `500 -> 250000`
- `502 -> 250200`
- `503 -> 250300`
- `1 -> 100100`
- `22 -> 102200`
- `-2 -> 110200`

## 6. 前端处理建议

- 仅根据响应体 `code` 处理，不依赖 HTTP 状态码。
- `240100` 统一视为未登录/登录失效。
- `240300` 统一视为无权限。
- 其他 `24xxxx` 视为标准错误分类；`10xxxx/11xxxx` 视为系统内部错误分类。

## 7. 当前已落地的子分类（末两位）

### 7.1 400 类（`2400xx`）

- `240001`：请求参数语义错误（如模板参数绑定非法）
- `240002`：参数值不支持（如枚举值非法）
- `240004`：缺少必要配置或必填信息

### 7.2 404 类（`2404xx`）

- `240401`：模板不存在（提示词模板/检索模板）
- `240402`：分析引擎不存在
- `240403`：实体不存在（文章/论坛/通用实体）
- `240405`：会话不存在或已结束
- `240406`：模型配置不存在
- `240407`：平台不存在
- `240408`：账号不存在或已删除
- `240409`：节点不存在
- `240410`：基础组件任务配置不存在
- `240411`：蓝图不存在
- `240412`：行动实例不存在
- `240413`：批注不存在
- `240414`：论坛不存在
- `240415`：文章不存在
- `240416`：沙盒不存在
- `240417`：节点实例不存在
- `240418`：节点定义不存在

### 7.3 409 类（`2409xx`）

- `240901`：资源已存在（主键或组合键冲突）
- `240902`：名称冲突
- `240903`：状态冲突（如已有待处理决策）

### 7.4 500 类（`2500xx`）

- `250000`：通用内部错误（兜底）
- `250001`：依赖未初始化（如 Elasticsearch）
- `250002`：查询或检索执行失败
- `250004`：业务执行失败（非依赖初始化问题）
- `250005`：沙盒生命周期操作失败

### 7.5 502/503 类映射扩展

- `250203`：外部嵌入服务调用失败
- `250301`：嵌入服务未初始化
